# RANSARA Keyboard
### Helakuru-style Singlish → සිංහල Unicode Keyboard for Android

---

## Features
- Full Singlish to Sinhala Unicode transliteration (Helakuru-style)
- All 54 Sinhala consonants + 18 vowel forms supported
- English mode toggle
- Shift key support
- Works in ANY Android app (WhatsApp, Facebook, Gmail, etc.)

## How to Build (Android Studio)

1. Install **Android Studio** (free): https://developer.android.com/studio
2. Open Android Studio → **Open Project** → select the `RANSARA_Keyboard` folder
3. Wait for Gradle sync to complete
4. Click **Build → Build Bundle(s)/APK(s) → Build APK(s)**
5. APK will be in: `app/build/outputs/apk/debug/app-debug.apk`
6. Copy APK to phone and install (enable "Install from unknown sources" in settings)

## How to Enable on Phone

1. Open **RANSARA Keyboard** app
2. Tap **"Step 1: Enable Keyboard"** → turn on RANSARA Keyboard
3. Tap **"Step 2: Select as Default"** → choose RANSARA Keyboard
4. Open any app and start typing!
